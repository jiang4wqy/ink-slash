# 一闪 · 墨斩 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 和风水墨风格的浏览器隔空切果游戏——指尖即刀,MediaPipe 手势 + 鼠标双输入,纯逻辑全单测。

**Architecture:** Vite+TS 无框架;`src/game/*` 与 `src/gesture/oneEuro.ts` 为纯函数/纯类(TDD),渲染(Canvas 2D)与音频(WebAudio 合成)为薄层经预览实测;主循环固定步长更新 + rAF 渲染;输入抽象 `BladePoint[]` 统一手势/鼠标。

**Tech Stack:** TypeScript(strict), Vite 7, Vitest 3, @mediapipe/tasks-vision(唯一运行时依赖)。

## Global Constraints

- 零图片/音频资产;字体栈 `"STKaiti","KaiTi","DFKai-SB","Noto Serif SC",serif`。
- 模型/wasm 由 `scripts/fetch-assets.mjs` postinstall 准备(下载 hand_landmarker.task → public/models;复制 wasm → public/wasm)。
- dev 端口 5178;localStorage 键前缀 `ink-slash:`。
- 经典规则数值:果 10 分;连击窗口 400ms,N≥2 加 N×10 分,N≥3 慢动作 600ms×0.35;3 命;炸弹概率 8%→15% 随分数线性(至 500 分封顶);速度门槛 600px/s×(viewportH/720)。
- 每个纯逻辑模块:先写失败测试 → 实现 → 通过 → 提交。
