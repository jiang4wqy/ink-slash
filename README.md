# 一闪 · 墨斩(Ink Slash)

宣纸上的居合斩。摄像头追踪你的手,**指尖即刀、墨迹为痕**:水墨画风的果实抛入画面,
隔空一挥,笔锋划过、果实应声而断,墨彩四溅在宣纸上留下渐淡的印渍。

一款完全在浏览器本地运行的手势体感切果游戏——**摄像头画面不上传、不保存**。
没有摄像头也可以用鼠标完整游玩。

## 玩法(幕制)

- **指尖(或鼠标)碰到果实即斩**——无需高速挥砍,轻轻掠过就切开;漏接**没有惩罚**。
- 游戏按**幕**推进:每幕一炷香(40 秒),香尽前斩满目标分即入下一幕;
  一幕比一幕更快更密,背景随幕流转**春→夏→秋→冬**。
- **终局只有两种**:一炷香烧尽还差分,或累计斩中 3 枚**铁炮弹**(画着「危」字,右上墨滴是余量)。
- 每果 10 分;**一挥连斩多果 = 连击**,朱红印章「N连斩」加成 N×10 分;三连斩以上触发慢动作。
- 第二幕起会随波次飘来**纸符道具**,斩开生效:
  「凍」时缓 3 秒(线香也一起变慢) · 「倍」双倍得分 6 秒 · 「墨」找回一滴墨。
- 双手可见时是**双刀**,指尖佩着一柄小小的水墨刀。菜单用指尖悬停按钮约 1 秒即可选择。
- **人影如墨**:摄像头模式下,你的身影会以淡墨晕染铺满宣纸——看得清自己、不遮挡游戏,
  且手与刀光在空间上 1:1 对齐;菜单里可切换 淡/浓/关。
- 最高分保存在本地浏览器中。

## 识别不灵? 试试这些

- **光线**:面向窗户或灯光,避免逆光(背后有强光会让手变成剪影)。
- **距离**:手离摄像头约 40–80cm,让整只手(含手掌)都在画面里——只伸进指尖是追踪不到的。
- **动作**:用手腕带动整只手划动,幅度大而干脆;超高速甩动会产生运动模糊,反而丢追踪。
- **背景**:身后越简洁越好,避免和肤色接近的杂物。
- 现在是**碰到即斩**,不需要用力快挥——从容划过即可。

## 运行

```powershell
npm install        # 安装依赖(自动下载手部模型与 wasm 到本地)
npm run dev        # 打开 http://127.0.0.1:5178
```

> Windows 用户也可直接双击 `启动.bat`,自动完成安装并打开浏览器。
>
> 摄像头需要安全上下文:请通过 `127.0.0.1` / `localhost` 访问。
> 若首次安装时未联网导致模型缺失,联网后运行 `npm run fetch-assets`。

## 构建与测试

```powershell
npm run build      # 类型检查 + 生产构建(纯静态,可任意托管)
npm run preview    # 预览构建产物
npm test           # Vitest:滤波/碰撞/物理/波次/计分/状态机 全部纯逻辑单测
```

## 技术

- **手势**:MediaPipe HandLandmarker(本地 wasm + 模型,GPU→CPU 自动回退),
  食指尖经 **One Euro 滤波**(快挥低延迟、慢移防抖),坐标按摄像头真实画幅比例映射。
- **判定**:两帧指尖连线做**扫掠线段碰撞**,30fps 下快挥不漏斩;速度门槛防误触。
- **画面**:Canvas 2D 程序化绘制——宣纸纹理、sumi-e 果实、收锋墨痕、飞白迸溅、
  朱印连击,**零图片资产**。
- **音效**:WebAudio 全合成(挥刀风声/太鼓/拨弦/钟声),**零音频资产**,可静音。
- 无框架、无运行时依赖(仅 @mediapipe/tasks-vision);构建产物约 150KB。
